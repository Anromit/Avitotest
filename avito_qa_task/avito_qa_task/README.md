# Тестовое задание для QA стажировки (Avito)

Решение тестового задания на позицию QA Trainee (Automation).
Стек: **Python + pytest + requests**.

## Описание
В репозитории находятся:
- `TESTCASES.md` — описание сценариев тестирования.
- `BUGS.md` — список несоответствий, найденных при тестировании API.
- `tests/` — код автотестов.

## Запуск тестов

1. Установка зависимостей:
   ```bash
   pip install -r requirements.txt
2. Запуск тестов:
    pytest -v
