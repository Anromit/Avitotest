# Тест-кейсы

**Host:** https://qa-internship.avito.com

| ID | Название | Шаги | Ожидаемый результат |
|----|----------|------|---------------------|
| 1 | Успешное создание объявления | 1. Отправить `POST /api/1/item` с корректными данными (sellerID, name, price, statistics). | Статус **200**. В ответе пришел JSON с полем `status` (текст успеха) и `id`. |
| 2 | Создание объявления с некорректной ценой | 1. Отправить `POST /api/1/item`.<br>2. В поле `price` передать строку "бесплатно". | Статус **400** (Bad Request). |
| 3 | Получение объявления по ID | 1. Создать объявление, сохранить `id`.<br>2. Отправить `GET /api/1/item/{id}`. | Статус **200**. Поля `name`, `price`, `sellerId` совпадают с отправленными. |
| 4 | Получение всех объявлений продавца | 1. Отправить `GET /api/1/{sellerID}/item`. | Статус **200**. В списке объявлений присутствует созданный нами товар. |
| 5 | Получение статистики | 1. Отправить `GET /api/1/statistic/{id}`. | Статус **200**. Поля статистики (`likes`, `viewCount`) совпадают с данными при создании. |
