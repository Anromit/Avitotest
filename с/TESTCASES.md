# Тест-кейсы для Таск-Трекера Авито

## Функциональные тест-кейсы

### 1. Создание задачи

**ID:** TC-001

**Приоритет:** High

**Предусловия:**
- Открыта главная страница таск-трекера
- Пользователь находится на вкладке "Список"

**Шаги:**
1. Нажать кнопку "Создать задачу"
2. Заполнить поле "Название" значением: "Тестовая задача ${текущая дата}"
3. Заполнить поле "Описание" значением: "Описание тестовой задачи"
4. Выбрать статус "В работе"
5. Нажать кнопку "Сохранить"

**Ожидаемый результат:**
- Задача успешно создана
- Задача отображается в списке задач
- У задачи указан выбранный статус
- Поля задачи соответствуют введенным данным

---

### 2. Открытие карточки задачи

**ID:** TC-002

**Приоритет:** Medium

**Предусловия:**
- В системе существует как минимум одна задача
- Пользователь находится на вкладке "Список"

**Шаги:**
1. Найти задачу в списке
2. Кликнуть на заголовок задачи

**Ожидаемый результат:**
- Открывается модальное окно с деталями задачи
- Отображаются все детали задачи (название, описание, статус)
- Доступны кнопки для редактирования и закрытия

---

### 3. Поиск задачи

**ID:** TC-003

**Приоритет:** High

**Предусловия:**
- В системе существуют задачи с разными названиями
- Пользователь находится на вкладке "Список"

**Шаги:**
1. В поле поиска ввести ключевое слово из названия существующей задачи
2. Нажать Enter или дождаться автоматического поиска

**Ожидаемый результат:**
- В результатах поиска отображаются только задачи, содержащие ключевое слово
- Несоответствующие задачи скрыты
- Отображается актуальный список задач

---

### 4. Перевод задач по статусам

**ID:** TC-004

**Приоритет:** High

**Предусловия:**
- Существует задача
- Открыта карточка задачи

**Шаги:**
1. Открыть карточку задачи
2. Изменить статус задачи через выпадающий список
3. Нажать кнопку "Сохранить"
4. Закрыть карточку

**Ожидаемый результат:**
- Статус задачи успешно изменен
- Новый статус отображается в списке задач
- Изменения сохраняются после обновления страницы

---

### 5. Переход на доску проекта

**ID:** TC-005

**Приоритет:** Medium

**Предусловия:**
- Пользователь находится на главной странице

**Шаги:**
1. Нажать на вкладку "Доска"
2. Дождаться загрузки доски

**Ожидаемый результат:**
- Открывается доска проекта с колонками статусов
- Задачи распределены по соответствующим колонкам статусов
- Отображается корректное количество задач в каждой колонке
