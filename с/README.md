# Автоматизированное тестирование Таск-Трекера

## Описание проекта

Автоматизированные тесты для проверки основных функций таск-трекера:
- Создание задач
- Просмотр карточек задач
- Поиск задач
- Управление статусами задач
- Навигация по доске проекта

## Технологический стек

- **Язык:** TypeScript/JavaScript
- **Фреймворк тестирования:** Playwright / Cypress
- **Запуск тестов:** npm scripts
- **Отчетность:** Allure Report / HTML Reporter

## Установка и настройка

### Предварительные требования

- Node.js 16+
- npm или yarn
- Браузер Chrome/Firefox

npm install
